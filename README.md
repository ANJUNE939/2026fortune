# Fortune Fate V11 Page Flow

요청사항 반영:
- PIN 입력 후 화면이 완전히 정보 입력 페이지로 전환
- 정보 입력 후 분석 선택 페이지로 전환
- 분석 메뉴 선택 후 결과 페이지로 전환
- STEP 진행 표시 추가
- 화면 전환 시 항상 최상단으로 이동
- 정보 입력 화면 상단에 '분석 정보 선택으로 이동' 버튼 추가
- 스크롤로 다음 STEP을 찾는 느낌을 줄이고 페이지가 바뀌는 방식으로 개선
- V10 캐시 제거 구조 유지
- V9 신뢰도 강화/자극적 답변 로직 유지
- Google Sheets 연동 유지
