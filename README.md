## GitHub Pages 배포 오류 해결

이 버전에는 `.nojekyll` 파일이 포함되어 있습니다.  
GitHub Pages가 Jekyll로 `backend/README.md` 등을 처리하다가 실패하는 문제를 막기 위한 파일입니다.

업로드 후 저장소 최상단에 `.nojekyll` 파일이 보여야 합니다.
